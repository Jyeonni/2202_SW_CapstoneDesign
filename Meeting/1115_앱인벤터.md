# 진행상황
#### 진행상황
- 앱인벤터 사용하여 로그인/로그아웃/회원가입/회원탈퇴 구현
- 키워드 입력 구현
  - 키워드 입력 텍스트와 한림대학교 공지사항 검색창 주소 join에 실패하여 검색시 오류발생
  - 한림대학교 공지사항 사이트를 웹뷰어로 보여주기 구현
