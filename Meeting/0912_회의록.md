# 프로젝트 계획
### 1. 어플리케이션 구현
- **앱 인벤터**로 결과물 완성하기
- 기한 : **2주** 이내
- 구동 순서 : 로그인 사이트 > 사용자가 키워드 입력 > 키워드가 포함된 공지사항 게시물 중 최근 50개 > 사용자에게 알림 전달
  - 날짜 지정, 날짜 정렬, 인기 검색순, 특정 단어 제외 등의 검색 옵션 추가 기능
  - 구글 검색창같은 검색창에 키워드 입력받기
- **역할 분담** : UI기획(다연, 서현) / 블록코딩구현(지원, 민홍)

### 2. 웹 구현
- 교수님 미팅 후 세부 역할 분담하기
