## 프로그램 설계
- 검색창에 사용자가 검색식을 씀
-> 유저의 string 검색식을 데베로 받아서 (방법 : 텍스트파일 등)
-> 데베에서 검색팀이 분해(파싱)
-> 유저별로 매칭
-> 공지사항 사이트에서 검색
-> 검색 결과 웹 팀에게 전달
-> 검색 결과 가공
-> 사용자에게 보여주기
* 로그인 하면 자동으로 텍스트파일(결과)을 보여주기
* 지난 번에 검색했던 결과를 보여주기

## 목표
#### 공통 : 교환하는 방법, 구현 방법 고민하기
- 웹팀에서 검색식을 줌
- 검색 결과 웹팀에 넘기기
- 주고받는 방법을 알아보기

#### 웹팀
- 로컬에서 가져온 게시물 보여주기(게시물 제목, 본문 보여주기) -> 100개 넘는 게시물 보여주는 방법 고민하기
- 로그인만 하면 검색결과를 보여주는 것(제일 중요)
- 고정IP 서버 만들기

#### 검색팀
- 검색식 만들기
- 검색결과 가져오기
- 그 결과를 가공해서 웹팀에서 보내주는 라이브러리 찾아보기
